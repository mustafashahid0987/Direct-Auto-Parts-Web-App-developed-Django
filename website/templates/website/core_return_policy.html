Core Return Policy

CORE RETURN

All remanufactured products are sold with an accompanying core charge. This core charge is refundable to the customer's account when the following criteria have been satisfied. All core returns must be:

  1. Received by Diesel Parts Direct within 30 days of the original invoice date
  2. Be rebuildable according to original equipment manufacturerâ€™s specifications (if deemed non rebuildable, core will be used for scrap or returned to customer via freight collect)
  3. The same part number that was provided to the customer
  4. Accompanied by a completed Core Return Form

Core Return Process

  1. Fill out our Core Return Form
  2. Print the form: it will include a shipping label and packing slip
  3. Pack and send your core
  4. Please pack your item carefully (remove grease and protect part from possible shipping damage)
  5. Place the packing slip inside your package with the core you are returning
  6. Place the shipping label outside of the package (make sure to include your return address and take note of your tracking number); the customer is responsible for shipping charges
  7. Your credit will be issued within 30 days from the day we received your core (if deemed rebuildable)
